## Summary

Signal Deck is a social ops cockpit for tracking TarotTALKS social media shares. Think "SHIP LOG, but for social." Log posts in under 15 seconds, filter and search your share history, and watch your social presence grow.

## Features

### Core Tracking
- **New Share Form**: Log a post in under 15 seconds
  - Platform selector (X, Bluesky, Threads, LinkedIn, Other)
  - Post URL with auto-resolution of shared TarotTALKS links
  - Auto-detects card/talk from pasted URLs (e.g., `tarottalks.app/cards/the-tower`)
  - Status tracking: Draft â†’ Posted â†’ Verified
  - Speaker handle capture for relationship tracking
  - Notes field for context

### Dashboard View
- **Stats Bar**: Today / This Week / Total share counts
- **Chronological Groups**: Today, Yesterday, This Week, Earlier
- **Share Rows**: Platform icon, title, status badge, external link

### Filters & Search
- Full-text search across notes and URLs
- Platform filter (X, Bluesky, Threads, LinkedIn, Other)
- Status filter (Draft, Posted, Verified)
- Date range picker

### URL Auto-Resolution
- Paste any TarotTALKS URL and it auto-detects:
  - Card pages â†’ links to card record
  - Talk pages â†’ links to talk record (with speaker name)
- API endpoint: `POST /api/admin/social-shares/resolve`

## Technical Implementation

### Database Schema
New `social_shares` table:
- `id` (UUID primary key)
- `platform` (enum: x, bluesky, threads, linkedin, other)
- `post_url` (the social platform URL)
- `status` (enum: draft, posted, verified)
- `posted_at` (timestamp)
- `shared_url` (the TarotTALKS URL being promoted)
- `card_id` / `talk_id` (foreign keys, nullable)
- `speaker_handle` / `speaker_name` (for relationship tracking)
- `notes` (freeform text)
- `created_at` / `updated_at`

### API Routes
- `GET /api/admin/social-shares` - List with filters + stats
- `POST /api/admin/social-shares` - Create new share
- `GET /api/admin/social-shares/[id]` - Get single share
- `PUT /api/admin/social-shares/[id]` - Update share
- `DELETE /api/admin/social-shares/[id]` - Delete share
- `POST /api/admin/social-shares/resolve` - Auto-resolve TarotTALKS URLs
